Folder Structure -> <br />
          -> NodeCrud <br />
                -> models <br />
                      -> testEntity.js <br />
                -> routes <br />
                      -> testEntities.js <br />
                -> app.js <br />
                -> package.js <br />
                -> package-lock.js <br />
                <br />
Models: Contains the schema for testEntity Entity <br />
Routes: Contains the api endpoints for the CRUD operations <br />
        Operations covered: GET, POST, PATCH <br />
app.js: Driver code
